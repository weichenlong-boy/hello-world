2020CCF企业非法集资风险预测

  下面来具体说一下这个赛题：

  首先是评价函数是F1，而catboost模型在这个题的表现比较好，是因为catboost模型的metric可以设置为F1，当然也可以使用lightgbm的自定义评估函数，效果差不多。我们采用的有5折交叉验证catboost模型，线上可以到0.8447，我另一个队友是使用lightgbm自定义损失函数和评估函数，线上也可以到达0.8425。

def custom_f1_eval(y_true, y_pred):
    y_pred_label = []
    for i in y_pred:
        if i<=0.5:
            y_pred_label.append(0)
        else:
            y_pred_label.append(1)
    f1_mean = f1_score(y_true, y_pred_label)    
    return "f1", f1_mean, True
def f1_loss(y, pred):
    beta = 2
    p = 1. / (1 + np.exp(-pred))
    grad = p * ((beta - 1) * y + 1) - beta * y
    hess = ((beta - 1) * y + 1) * p * (1.0 - p)
    return grad, hess

  其次是特征工程，因为赛题的超级玄学，我们并没有做太多特征(其实是做不上去)。和其他结构化比赛差不多的一些操作，特征交叉，特征编码，时间特征，groupby特征等一些常规操作。这里就不展示了，有兴趣的同学可以在github上查看。

  接下来就讲一讲本次比赛baseline的地方，水哥的baseline地址：

https://github.com/datawhalechina/competition-baseline/tree/master/competition/DataFountain-%E4%BC%81%E4%B8%9A%E9%9D%9E%E6%B3%95%E9%9B%86%E8%B5%84%E9%A3%8E%E9%99%A9%E9%A2%84%E6%B5%8B

  我通过对lightgbm模型每次使用不同的种子，然后随机搜索参数100轮，得到每个种子的最优参数组，然后随机生成100个种子，这样下来就得到了100组最优参数。然后将这些参数随机选择带入模型融合，线上可以有不错的效果，可以达到8486，相对于baseline提高了5个千分点。通过这种操作得到两个线上的高分文件，分别为8472和8486。

  我们最后的操作就是经典模型融合，首先对catboost和lightgbm得出的三个文件进行stacking。

xlf=xgb.XGBClassifier(max_depth=5,learning_rate=0.05,n_estimators=55,reg_alpha=0.005,n_jobs=8,random_state=2020,importance_type='total_cover')
